# Cautalabs Food Model

A multimodal AI-powered food safety and quality sensing prototype.

This project combines:
- NIR multispectral imaging
- VIS+NIR spectroscopy
- VOC/gas sensing
- Depth (ToF) sensing
- Thermal sensing
- On-device AI inference
