# Parts List (Current Prototype)

## Compute
- Raspberry Pi 5 (8GB)
- Raspberry Pi AI HAT (26 TOPS)

## Imaging
- Raspberry Pi Camera Module 3 NoIR
- Raspberry Pi Camera Module 3 (RGB)

## Illumination
- 3W LED – 730 nm
- 3W LED – 850 nm
- 3W LED – 940 nm
- White LED (visible illumination)

## Sensors
- BME690 (VOC, temperature, humidity, pressure)
- AS7343 spectral sensor
- VL53L8CX Time-of-Flight sensor
- MLX90614 (GY-906-BCC) infrared temperature sensor

## Power & Control
- LM2596 CC/CV DC-DC modules (x3)
- MOSFET trigger modules
- RTC battery module
- 0.96" OLED display (I2C)
- 5V DC fan (VOC chamber)
- Momentary push button with LED ring
  - Used as manual acquisition trigger
  - Switch contacts connected to Raspberry Pi GPIO


## Mechanical / Optical
- 60° collimator lenses
- Lens holders
- Protective lens filter
